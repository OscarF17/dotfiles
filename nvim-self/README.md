# Neovim configuration
This is my personal Neovim setup. I originally used [AstroNvim](https://astronvim.com/) with slight modifications (also avaliable on [this repo](https://github.com/OscarF17/dotfiles/tree/main/astronvim)), but felt like creating my own from scratch. My setup shamelessly borrows a lot of ideas from AstroNvim, such as hotkeys and plugins, so it should provide a somewhat familiar vibe.


## Dependencies
Each plugin file already installs its necessary plugin dependencies, but external system dependencies are still needed. This list may not contain all of them, check the GitHub page for plugins that present issues.

- ripgrep
- lazygit
- Nerd font and nerd-font-compatible terminal
